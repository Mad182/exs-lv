Skaidrojums

Šajā sadaļā aprakstīti tie pieprasījumi, kas neietilpst citās sadaļās.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/random/.

Lai izpildītu kādu no pieprasījumiem, lietotājam jābūt autentificētam.

Lietotāja notikumi

Atgriež sarakstu ar 25 (maināma vērtība) jaunākajiem lietotāja notikumiem.

Adrese: https://ios.exs.lv/random/notifications

Atbilde:

json
{
    "status": 200,
    "response": {
        "notifications": [
            {
                "id": int,
                "type": int,
                "group_id": int,
                "foreign_key": int,
                "text": string,
                "date": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
notifications masīvs ar objektiem par jaunākajiem notikumiem
notifications : id notikuma ID
notifications : type notikuma paveids (skat. nākamo tabulu)
notifications : group_id grupas ID, ja notikums saistīts ar kādu domubiedru grupu
notifications : foreign_key ID, kas saistīts ar konkrēto notikumu un dažādiem notikumu tipiem apzīmē ko citu (skat. tabulu)
notifications : text īss notikuma teksts, piemēram, fragments no minibloga teksta.
Neskaitot miniblogus, citiem notikumiem pārsvarā tukšs string.
notifications : date notikuma parādīšanās laiks teksta formātā, piemēram, "pirms 3 mēnešiem"

Tiešsaistē esošie lietotāji

Atgriež sarakstu ar tiešsaistē esošajiem lietotājiem pēdējās 5 minūtēs.

Adrese: https://ios.exs.lv/random/online

Atbilde:

json
{
    "status": 200,
    "response": {
        "online": int,
        "registered": int,
        "users": [
            {
                "nick": string,
                "params": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
online tiešsaistē esošo lietotāju skaits
registered tiešsaistē esošo lietotāju skaits, kas ir autentificējušies
users masīvs ar objektiem par tiešsaistē esošajiem autentificētiem lietotājiem
users : nick lietotāja lietotājvārds
users : params vairākas vērtības, atdalītas ar "|", kas raksturo lietotājvārdu, lai to varētu pareizi noformatēt, piemēram, "115|2|1|3|0":
1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir
