Skaidrojums

Šajā sadaļā aprakstīti tie pieprasījumi, kam ir saistība ar domubiedru grupām.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/groups/.

Lai izpildītu kādu no pieprasījumiem, lietotājam jābūt autentificētam.

Lietotāja grupu saraksts

Atgriež sarakstu ar grupām, kurās lietotājs ir pieteicies, un nelasīto ziņu skaitu tajās.

Adrese: https://ios.exs.lv/groups/group_news

Atbilde:

json
{
    "status": 200,
    "response": {
        "group_count": int,
        "groups": [
            {
                "id": int,
                "avatar_url": string,
                "title": string,
                "member_count": int,
                "post_count": int,
                "is_admin": bool,
                "is_mod": bool,
                "unread_msgs": int
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
group_count grupu skaits, kurās lietotājs ir pieteicies vai kuras ir izveidojis
groups saraksts ar grupām
groups : id grupas ID
groups : avaar_url saite uz grupas avataru
groups : title grupas nosaukums
groups : member_count grupas biedru skaits, ieskaitot grupas autoru
groups : post_count grupā veikto ierakstu skaits
groups : is_admin vai autentificētais lietotājs šajā grupā ir administrators
groups : is_mod vai autentificētais lietotājs šajā grupā ir moderators
groups : unread_msgs autentificētā lietotāja neizlasīto ierakstu skaits šajā grupā

Kategorijas grupu saraksts

Atgriež sarakstu ar grupām norādītajā grupu kategorijā.

Adrese: https://ios.exs.lv/groups/cat_groups/{cat_id}?page={page_nr}

{cat_id} - grupu kategorijas ID.
{page_nr} - grupu saraksta lappuse. Parametru nenorādot, atgriež pirmo lappusi.

Atbilde:

json
{
    "status": 200,
    "response": {
        "group_count": int,
        "page_count": int,
        "current_page": int,
        "per_page": int,
        "category_id": int,
        "category_title": string,
        "groups": [
            {
                "id": int,
                "avatar_url": string,
                "title": string,
                "member_count": int,
                "post_count": int,
                "in_group": bool,
                "is_admin": bool,
                "is_mod": bool,
                "unread_msgs": int
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
group_count grupu skaits šajā kategorijā
page_count šīs kategorijas grupu saraksta lappušu skaits
current_page pieprasītā lappuse
per_page vienā lappusē atgriezto grupu skaits (maināma vērtība)
category_id grupu kategorijas ID
category_title grupu kategorijas nosaukums
groups masīvs ar objektiem par kategorijas grupām
groups : in_group vai autentificētais lietotājs ir pieteicies šajā grupā?

Informācija par grupu

Atgriež informāciju par grupu, tai skaitā grupas sākumlapas cilnes saturu.

Adrese: https://ios.exs.lv/groups/{group_id}/home

Atbilde:

json
{
    "status": 200,
    "response": {
        "group_data": {
            "category_id": int,
            "category_title": string,
            "group_id": int,
            "group_title": string,
            "intro_text": string,
            "image_count": int,
            "image_urls": string[],
            "avatar_url": string,
            "member_count": int,
            "post_count": int,
            "posts_seen": int,
            "is_member": bool,
            "created_by": {
                "nick": string,
                "params": string,
                "avatar_url": string
            },
            "is_archived": bool
        }
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
category_id ID grupu kategorijai, pie kuras pieder šī grupa
category_title nosaukums grupu kategorijai, pie kuras pieder šī grupa
group_id grupas ID
group_title grupas nosaukums
intro_text grupas sākumlapas cilnes saturs;
smaidiņi atstāti kā simboli;
attēli aizstāti ar tektu "(Attēls #<nr>)", kur <nr> ir attēla kārtas nr. tekstā;
spoileri aizstāti ar tekstu "(spoileris slēpts)";
widgeti (twitter, youtube utt.) atstāti kā saites;
pārveidotas adreses: "<a href="https://exs.lv">spied šeit</a>" uz "https://exs.lv";
atstāti tikai šādi HTML tagi: <a><img><br><p><strong><b><i><em>
image_count tekstā atrasto attēlu skaits (sakrīt ar parametrā "image_urls" esošā masīva izmēru)
image_urls masīvs ar saitēm uz tekstā atrastajiem attēliem
avatar_url saite uz grupas avataru
member_count grupas biedru skaits, ieskaitot grupas autoru
post_count ierakstu skaits grupā
posts_seen autentificētā lietotāja redzēto ierakstu skaits šajā grupā (pēc pievienošanas datuma)
is_member vai autentificētais lietotājs ir šīs grupas biedrs?
created_by objekts ar informāciju par grupas autoru
created_by : nick grupas autora lietotājvārds
created_by : params vairākas vērtības, atdalītas ar "|", kas raksturo lietotājvārdu, lai to varētu pareizi noformatēt, piemēram, "115|2|1|3|0":
1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir
created_by : avatar_url saite uz grupas autora avataru
is_archived vai šī grupa ir arhivēta?

Grupas lietotāju cilne

Atgriež sarakstu ar grupā esošajiem lietotājiem.

Adrese: https://ios.exs.lv/groups/{group_id}/members?page={nr}

{nr} - biedru saraksta lappuse. Parametru nenorādot, tiks atgriezta pirmā lappuse.

Atbilde:

json
{
    "status": 200,
    "response": {
        "member_count": int,
        "page_count": int,
        "current_page": int,
        "per_page": int,
        "members": [
            {
                "member_id": int,
                "is_admin": bool,
                "is_mod": bool,
                "nick": string,
                "params": string,
                "avatar_url": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
member_count biedru skaits grupā, ieskaitot grupas administratoru
page_count biedru saraksta lappušu skaits
current_page pieprasītā lappuse
per_page vienā lappusē atgriezto biedru skaits (maināma vērtība)
members masīvs ar objektiem par grupas biedriem
members : member_id grupas biedra ID (nav tas pats, kas lietotāja ID!)
members : is_admin vai biedrs ir grupas administrators?
members : is_mod vai biedrs ir grupas moderators?
members : nick grupas biedra lietotājvārds
members : params detalizētāka informācija par grupas biedra lietotājvārda formatējumu (piemēram, "115|2|1|3|0")
members : avatar_url saite uz biedra avataru

json
{
    "status": 200,
    "response": {
        "page_count": int,
        "current_page": int,
        "per_page": int,

            {
                "id": int,
                "short_text": string,
                "author": {                
                    "nick": string,
                    "params": string,
                    "avatar_url": string
                },
                "created_at": string,
                "bumped_at": string,
                "post_count": int,
                "interface": int,
                "is_closed": bool
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Parametrs Skaidrojums
page_count kopumā skatāmo miniblogu lappušu skaits (maināma vērtība)
current_page pieprasītā lappuse
per_page vienā lappusē atgriezto miniblogu skaits (maināma vērtība)
miniblogs masīvs ar jaunāko miniblogu objektiem
miniblogs : id minibloga ID

miniblogs : author objekts ar informāciju par minibloga autoru

1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir
miniblogs : author: avatar_url pilna adrese uz lietotāja avataru

formāta piemērs: "2016-07-11 21:21:53"

miniblogs : is_closed vai miniblogs jau ir aizvērts?

Jauna grupas minibloga/komentāra pievienošana

Adrese: https://ios.exs.lv/groups/{new|comment}?token={token}

{new|comment} - atbilstoši jaunam miniblogam vai minibloga komentāram/atbildei.
{token} - pie autentificēšanās saņemtā vērtība atslēgā "token".

Šis ir POST tipa pieprasījums, un tam jāpievieno šādas POST vērtības:

Parametrs Vērtības tips Skaidrojums
group_id int grupas ID
parent_id int pievienojamā ieraksta parent ID;
pievienojot jaunu miniblogu - 0;
pievienojot komentāru - minibloga ID;
pievienojot atbildi - komentāra ID
mb_text string pievienojamā minibloga/komentāra/atbildes teksts

Atbilde:

json
{
    "status": 200,
    "response": {
        "miniblog_id": int
    }
}

Atbildes vērtība skaidrota tabulā:

Atslēga Skaidrojums
miniblog_id ID miniblogam, kas nupat izveidots vai kuram pievienots komentārs/atbilde

Grupas minibloga vērtēšana

Ļauj novērtēt miniblogu vai tā komentāru/atbildi ar plusiņu vai mīnusiņu.

Adrese: https://ios.exs.lv/groups/{plus|minus}/{mb_id}?token={token}

{plus|minus} - atbilstoši pozitīvam vai negatīvam vērtējumam.
{mb_id} - minibloga/komentāra/atbildes ID.
{token} - pie autentificēšanās saņemtā vērtība atslēgā "token".

Atbilde:

json
{
    "status": 200,
    "response": {
        "vote_value": int
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
vote_value ieraksta jaunais vērtējums pēc izpildīta pieprasījuma

Lietotāja pieteikšanās grupai

Ļauj lietotājam pieteikties par biedru kādā no grupām.

Adrese: https://ios.exs.lv/groups/{group_id}/apply?token={token}

{group_id} - ID grupai, kurā lietotājs vēlas kļūt par biedru.
{token} - pie autentificēšanās saņemtā vērtība atslēgā "token".

Atbilde:

json
{
    "status": 200,
    "response": {
        "joined": bool
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
joined vērtība vienmēr būs vienāda ar TRUE (ja vien netiks atgriezta kāda kļūda parametrā "success" un "message", tādējādi šo parametru nemaz nepievienojot)

Lietotāja izstāšanās no grupas

Ļauj lietotājam izstāties no grupas.

Adrese: https://ios.exs.lv/groups/{group_id}/leave?token={token}

{group_id} - ID grupai, no kuras lietotājs vēlas izstāties.
{token} - pie autentificēšanās saņemtā vērtība atslēgā "token".

Atbilde:

json
{
    "status": 200,
    "response": {
        "left": bool
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
left vērtība vienmēr būs vienāda ar TRUE (ja vien netiks atgriezta kāda kļūda parametrā "success" un "message", tādējādi šo parametru nemaz nepievienojot)
