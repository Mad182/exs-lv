Skaidrojums

Šajā sadaļā aprakstīti ar attēlu kolekcijām saistīti pieprasījumi.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/collections/.

Lai kādu no tiem izpildītu, lietotājam NAV jābūt autentificētam.
IP lieguma gadījumā šie pieprasījumi nebūs pieejami.

Smaidiņu saraksts

Atgriež sarakstu ar exs.lv izmantotajiem smaidiņiem un adresēm uz to attēliem.

Adrese: https://ios.exs.lv/collections/smilies

Atbilde:

json
{
    "status": 200,
    "response": {
        "smilies_count": int,
        "smilies": [
            {
                "code": string,
                "title": string,
                "url": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
smilies_count smaidiņu skaits parametrā "smilies"
smilies masīvs ar objektiem par smaidiņiem
smilies : code smaidiņa kods, kas izmantojams tekstā
smilies : title smaidiņa attēla nosaukums, ieskaitot attēla formātu
smilies : url pilna adrese uz smaidiņa attēlu

Notifikāciju attēlu saraksts

Atgriež sarakstu ar attēliem, kādi tiek izmantoti lietotāja notifikācijām.

Adrese: https://ios.exs.lv/collections/notifications

Atbilde:

json
{
    "status": 200,
    "response": {
        "list_size": int,
        "notifications": [
            {
                "type": int,
                "title": string,
                "url": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
list_size elementu skaits parametrā "notifications"
notifications : type notifikācijas tips;
sīkāku tipu skaidrojumu meklēt šeit.
notifications : title notifikācijas nosaukums
notifications : url pilna adrese uz notifikācijas attēlu
