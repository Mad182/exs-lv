Jaunākās API izmaiņas

24.01.2017.

  • pie 441 statusa atbildes klāt tiek pievienoti arī lietotāja profila dati, piemēram, lietotājvārds un avatara adrese.

23.07.2016.

  • mainīts API atbilžu formāts, kas aprakstīts sadaļā "Ievads";
  • XSRF atslēga adrešu galā pārsaukta par "token";
  • API iekļautas divu-soļu-autentifikācijas pārbaudes, kas aprakstītas sadaļā "Autentifikācija";
  • sadaļā "Autentifikācija" pievienots pieprasījums, kas ļauj atkārtoti iegūt "token" vērtību;
  • /ban_details adrese mainīta uz /statuses/ban_details.

22.07.2016.

  • sadaļā "Kolekcijas" pieejamie pieprasījumi turpmāk būs izpildāmi arī tad, ja lietotājs nebūs autentificējies.
  • izveidota API sadaļa "Autentifikācija";
  • autentificēšanās un izlogošanās darbības pārceltas uz sadaļu "Autentifikācija", mainoties arī adresei;
  • izlogošanās pieprasījuma adresē turpmāk jānorāda XSRF atslēga.

14.07.2016.

  • pievienota API sadaļa "kolekcijas";
  • saraksts ar smaidiņiem kolekciju sadaļā;
  • saraksts ar notifikāciju attēliem kolekciju sadaļā.

13.07.2016.

  • globālas izmaiņas daudzos pieprasījumu veidos;
  • vairāki ar grupām saistīti pieprasījumi pārvietoti no "Dažādi" uz "Grupas", mainoties arī adresei;
  • pievienota profilu API sadaļa;
  • uz profilu sadaļu pārvietots zem "Dažādi" atrodamais pieprasījums par lietotāja profilu.
