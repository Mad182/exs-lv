Skaidrojums

Šajā sadaļā aprakstīti tie pieprasījumi, kas saistīti ar vēstulēm.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/inbox/.

Lai izpildītu kādu no pieprasījumiem, lietotājam jābūt autentificētam.

Saņemtās vēstules

Atgriež sarakstu ar saņemtajām vēstulēm.

Adrese: https://ios.exs.lv/inbox/received?page={page_nr}

{page_nr} - vēstuļu saraksta lappuse. Parametru nenorādot, atgriež pirmo lappusi.

Atbilde:

json
{
    ...,
    "response": {
        "endoflist": bool,
        "unread": int,
        "messages": [
            {
                "id": int,
                "title": string,
                "date": string,
                "from": {
                    "nick": string,
                    "params": string,
                    "avatar_url": string
                },
                "is_read": bool
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
endoflist sasniedzot pēdējo skatāmo vēstuļu saraksta lappusi, atgriezīs TRUE, signalizējot, ka vairāk saņemtu vēstuļu nav
unread vēl neizlasīto vēstuļu skaits
messages masīvs ar objektiem par saņemtajām vēstulēm
messages : id vēstules ID
messages : title vēstules virsraksts
messages : date laiks, kad vēstule saņemta (piemēram, "Šodien, 21:32")
messages : from vēstules autora dati
messages : from : nick vēstules autora lietotājvārds
messages : from : params vairākas vērtības, atdalītas ar "|", kas raksturo lietotājvārdu, lai to varētu pareizi noformatēt, piemēram, "115|2|1|3|0":
1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir
messages : from : avatar_url saite uz vēstules autora avataru
messages : is_read vai vēstule ir izlasīta?

  •    - skatāmo lappušu skaits nav ierobežots
  •    - katrā lappusē tiek atgriezta informācija par 20 vēstulēm (maināma vērtība)

Jauna vēstule

Ļauj nosūtīt kādam lietotājam jaunu vēstuli.

Adrese: https://ios.exs.lv/inbox/send

Šis ir POST tipa pieprasījums, un tam jāpievieno šādas POST vērtības:

Parametrs Vērtības tips Skaidrojums
msg_to int vēstules saņēmēja lietotāja ID
msg_title string vēstules virsraksts
msg_content string vēstules saturs

Atbilde:

json
{
    ...,
    "response": {
        "sent": bool
    }
}

Atbildes vērtība skaidrota tabulā:

Atslēga Skaidrojums
sent vienmēr vienāda ar TRUE (ja vien netiks atgriezta kāda kļūda parametrā "success" un "message", tādējādi šo parametru nemaz nepievienojot)

Vēstules lasīšana

Atgriež informāciju par norādīto vēstuli.

Adrese: https://ios.exs.lv/inbox/read/{message_id}

{message_id} - lasāmās vēstules ID

Atbilde:

json
{
    ...,
    "response": {
        "message": {
            "id": int,
            "title": string,
            "text": string,
            "image_count": int,
            "image_urls": string[],
            "datetime": string,
            "user": {
                "nick": string,
                "params": string,
                "avatar_url": string
            }
        }
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
message informācija par vēstuli
message : id vēstules ID
message : title vēstules virsraksts
message : text vēstules teksts;
smaidiņi atstāsti kā simboli;
attēli aizstāti ar tektu "(Attēls #<nr>)", kur <nr> ir attēla kārtas nr. tekstā;
spoileri aizstāti ar tekstu "(spoileris slēpts)";
widgeti (twitter, youtube utt.) atstāti kā saites;
pārveidotas adreses: "<a href="https://exs.lv">spied šeit</a>" uz "https://exs.lv";
atstāti tikai šādi HTML tagi: <a><img><br><p><strong><b><i><em>
message : image_count tekstā atrasto attēlu skaits (sakrīt ar parametrā "image_urls" esošā masīva izmēru)
message : image_urls masīvs ar saitēm uz tekstā atrastajiem attēliem
message : datetime laiks teksta formātā: "2016-07-10 21:40"
message : user vēstules saņēmēja/adresāta dati
