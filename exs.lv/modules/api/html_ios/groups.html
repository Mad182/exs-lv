Skaidrojums

Šajā sadaļā aprakstīti tie pieprasījumi, kam ir saistība ar domubiedru grupām.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/groups/.

Lai izpildītu kādu no pieprasījumiem, lietotājam jābūt autentificētam.

Informācija par grupu

Atgriež informāciju par grupu, tai skaitā grupas sākumlapas cilnes saturu.

Adrese: https://ios.exs.lv/groups/{group_id}/home

Atbilde:

json
{
    ...,
    "response": {
        "group_data": {
            "category_id": int,
            "category_title": string,
            "group_id": int,
            "group_title": string,
            "intro_text": string,
            "image_count": int,
            "image_urls": string[],
            "avatar_url": string,
            "member_count": int,
            "post_count": int,
            "posts_seen": int,
            "owner_data": {
                "nick": string,
                "params": string,
                "avatar_url": string
            },
            "is_member": bool
            "is_archived": bool
        }
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
category_id ID grupu kategorijai, pie kuras pieder šī grupa
category_title nosaukums grupu kategorijai, pie kuras pieder šī grupa
group_id grupas ID
group_title grupas nosaukums
intro_text grupas sākumlapas cilnes saturs;
smaidiņi atstāti kā simboli;
attēli aizstāti ar tektu "(Attēls #<nr>)", kur <nr> ir attēla kārtas nr. tekstā;
spoileri aizstāti ar tekstu "(spoileris slēpts)";
widgeti (twitter, youtube utt.) atstāti kā saites;
pārveidotas adreses: "<a href="https://exs.lv">spied šeit</a>" uz "https://exs.lv";
atstāti tikai šādi HTML tagi: <a><img><br><p><strong><b><i><em>
image_count tekstā atrasto attēlu skaits (sakrīt ar parametrā "image_urls" esošā masīva izmēru)
image_urls masīvs ar saitēm uz tekstā atrastajiem attēliem
avatar_url saite uz grupas avataru
member_count grupas biedru skaits, ieskaitot grupas autoru
post_count ierakstu skaits grupā
posts_seen autentificētā lietotāja redzēto ierakstu skaits šajā grupā (pēc pievienošanas datuma)
owner_data objekts ar informāciju par grupas autoru
owner_data : nick grupas autora lietotājvārds
owner_data : params vairākas vērtības, atdalītas ar "|", kas raksturo lietotājvārdu, lai to varētu pareizi noformatēt, piemēram, "115|2|1|3|0":
1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir
owner_data : avatar_url saite uz grupas autora avataru
is_member vai autentificētais lietotājs ir šīs grupas biedrs?
is_archived vai šī grupa ir arhivēta?

Grupas lietotāju cilne

Atgriež sarakstu ar grupā esošajiem lietotājiem.

Adrese: https://ios.exs.lv/groups/{group_id}/members?page={nr}

{nr} - biedru saraksta lappuse. Parametru nenorādot, tiks atgriezta pirmā lappuse.

Atbilde:

json
{
    ...,
    "response": {
        "group_members": [
            {
                "member_id": int,
                "is_admin": bool,
                "is_mod": bool,
                "avatar_url": string,
                "nick": string,
                "params": string
            },
            ...
        ],
        "endoflist": bool
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
group_members masīvs ar objektiem par grupas biedriem
group_members : obj : member_id grupas biedra ID (nav tas pats, kas lietotāja ID!)
group_members : obj : is_admin vai biedrs ir grupas administrators?
group_members : obj : is_mod vai biedrs ir grupas moderators?
group_members : obj : avatar_url saite uz biedra avataru
group_members : obj : nick grupas biedra lietotājvārds
group_members : obj : params detalizētāka informācija par grupas biedra lietotājvārda formatējumu (piemēram, "115|2|1|3|0")
endoflist sasniedzot pēdējo skatāmo biedru saraksta lappusi, atgriezīs TRUE, signalizējot, ka vairāk biedru nav

  •    - skatāmo lappušu skaits nav ierobežots
  •    - katrā lappusē tiek atgriezta informācija par 20 biedriem (maināma vērtība)
  •    - saraksta 1. lappuse nekad nebūs tukša, jo tajā vienmēr būs vismaz grupas autors

Grupas miniblogu cilne

Atgriež sarakstu ar jaunākajiem grupas miniblogiem.

Adrese: https://ios.exs.lv/groups/{group_id}/getlatestlist?page={page_nr}

{group_id} - grupas ID
{page_nr} - skatāmā miniblogu saraksta lappuse. Parametru nenorādot, tiks atgriezta pirmā lappuse.

Atbilde:

json
{
    ...,
    "response": {
        "miniblogs": [
            {
                "id": int,
                "short_text": string,
                "author": {                
                    "nick": string,
                    "params": string
                    "avatar_url": string
                },
                "time_before": string;
                "post_count": int,
                "is_closed": bool,
                "closed_by": 0|{
                    "nick": string,
                    "params": string
                },
                "close_reason": string
            }
            ...
        ],
        "endoflist": bool
    }
}

Atbildes vērtības skaidrotas tabulā:

Parametrs Skaidrojums
miniblogs masīvs ar jaunāko grupas miniblogu objektiem
miniblogs : id grupas minibloga ID
miniblogs : short_text saīsināts grupas minibloga satura teksts max. 300 simbolu garumā ar daudzpunkti galā (ja satura vairāk); attēli kā saites, spoileri nodzēsti, smaidiņi kā simboli, bez HTML tagiem
miniblogs : author objekts ar informāciju par grupas minibloga autoru
miniblogs : time_before laiks, pirms kura grupas miniblogā veikts jaunākais ieraksts
(piemēram: "pirms 3 nedēļām")
miniblogs : post_count komentāru skaits grupas miniblogā
miniblogs : is_closed vai grupas miniblogs jau ir aizvērts?
miniblogs : closed_by objekts ar informāciju par lietotāju, kas grupas miniblogu aizslēdzis, vai 0
miniblogs : close_reason grupas minibloga aizvēršanas iemesls
endoflist sasniedzot pēdējo skatāmo lappusi, atgriezīs TRUE, signalizējot, ka vairāk parādāmo ierakstu nav

  •    - skatāmas ne vairāk kā jaunākās 10 lappuses (maināma vērtība)
  •    - katrā lappusē tiek atgriezta informācija par 20 miniblogiem (maināma vērtība)

Jauna grupas minibloga/komentāra pievienošana

Adrese: https://ios.exs.lv/groups/{new|comment}

{new|comment} - atbilstoši jaunam miniblogam vai minibloga komentāram/atbildei

Šis ir POST tipa pieprasījums, un tam jāpievieno šādas POST vērtības:

Parametrs Vērtības tips Skaidrojums
group_id int grupas ID
parent_id int pievienojamā ieraksta parent ID;
pievienojot jaunu miniblogu - 0;
pievienojot komentāru - minibloga ID;
pievienojot atbildi - komentāra ID
mb_text string pievienojamā minibloga/komentāra/atbildes teksts

Atbilde:

json
{
    ...,
    "response": {
        "miniblog_id": int
    }
}

Atbildes vērtība skaidrota tabulā:

Atslēga Skaidrojums
miniblog_id ID miniblogam, kas nupat izveidots vai kuram pievienots komentārs/atbilde

Grupas minibloga ielāde

Atgriež informāciju par norādīto grupas miniblogu.

Adrese: https://ios.exs.lv/groups/getminiblog/{mb_id}

{mb_id} - grupas minibloga ID

Atbilde:

json
{
    ...,
    "response": {
        "miniblog": {
            "id": int,
            "full_text": string,
            "image_count": int,
            "image_urls": string[],
            "author": {                
                "nick": string,
                "params": string,
                "avatar_url": string
            },
            "datetime": string,
            "vote_value": int,
            "has_voted": bool,
            "is_closed": bool,
            "closed_by": 0|{
                "nick": string,
                "params": string,
                "avatar_url": string
            },
            "close_reason": string,
            "group_data": false|{
                "id": int,
                "title": string,
                "avatar_url": string
            }
        },
        "comments": [
            "0": [
                {
                    "id": int,
                    "full_text": string,
                    "author": {                
                        "nick": string,
                        "params": string,
                        "avatar_url": string
                    },
                    "datetime": string,
                    "is_removed": int,
                    "vote_value": int,
                    "has_voted": bool,
                    "image_urls": string[],
                    "image_count": int
                },
                ...
            ],
            int : [
                {
                    ...
                },
                ...
            ],
            ...,
            "comment_count": int
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Parametrs Skaidrojums
miniblog objekts ar informāciju par miniblogu
miniblog : id minibloga ID
miniblog : full_text minibloga teksts;
smaidiņi atstāsti kā simboli;
attēli aizstāti ar tektu "(Attēls #<nr>)", kur <nr> ir attēla kārtas nr. tekstā;
spoileri aizstāti ar tekstu "(spoileris slēpts)";
widgeti (twitter, youtube utt.) atstāti kā saites;
pārveidotas adreses: "<a href="https://exs.lv">spied šeit</a>" uz "https://exs.lv";
atstāti tikai šādi HTML tagi: <a><img><br><p><strong><b><i><em>
miniblog : image_count tekstā atrasto attēlu skaits (sakrīt ar parametrā "image_urls" esošā masīva izmēru)
miniblog : image_urls masīvs ar saitēm uz tekstā atrastajiem attēliem
miniblog : author objekts ar informāciju par minibloga autoru
miniblog : datetime laiks, kad ieraksts pievienots, piemēram, "19.03.2015. 21:28"
miniblog : vote_value minibloga vērtējums;
pozitīva vērtējuma gadījumā "+" zīme netiek pierakstīta
miniblog : has_voted vai autentificētais lietotājs šo miniblogu jau ir novērtējis
miniblog : is_closed vai miniblogs ir slēgts komentēšanai?
miniblog : closed_by objekts ar informāciju par lietotāju, kas miniblogu aizslēdzis, vai 0
miniblog : close_reason minibloga aizvēršanas iemesls
miniblog : group_data informācija par grupu, kurai pievienots miniblogs, vai FALSE
miniblog : group_data : id ID grupai, kurā pievienots šis miniblogs, vai 0
miniblog : group_data : title nosaukums grupai, kurā pievienots šis miniblogs, vai tukšs string
miniblog : group_data : avatar_url saite uz avataru grupai, kurā pievienots šis miniblogs, vai tukšs string
comments masīvs ar miniblogam pievienoto komentāru/atbilžu objektiem
comments : int skaitliska vērtība, kas apzīmē komentāra parent ieraksta ID;
zem 0 - masīvs ar minibloga 1. līmeņa komentāriem;
visi pārējie ID ir kāda 1. līmeņa komentāra ID, kas nozīmē, ka katrs no tiem satur masīvu ar atbildēm uz šo 1. līmeņa komentāru
comments : comment_count miniblogam pievienoto komentāru skaits (ieskaitot komentāru atbildes)
comments : int : id komentāra ID
comments : int : full_text komentāra/atbildes teksts;
formatēts pēc tāda paša principa kā paša minibloga teksts
comments : int : author objekts ar informāciju par komentāra autoru
comments : int : datetime laiks teksta formātā: "pirms ..."
comments : int : is_removed 0 - ieraksts nav dzēsts;
1 - ieraksts dzēsts
comments : int : vote_value komentāra vērtējums;
pozitīva vērtējuma gadījumā "+" zīme netiek pierakstīta
comments : int : has_voted vai autentificētais lietotājs šo komentāru jau ir novērtējis?
comments : int : image_urls masīvs ar saitēm uz tekstā atrastajiem attēliem
comments : int : image_count tekstā atrasto attēlu skaits (sakrīt ar parametrā "image_urls" esošā masīva izmēru)

Grupas minibloga vērtēšana

Ļauj novērtēt miniblogu vai tā komentāru/atbildi ar plusiņu vai mīnusiņu.

Adrese: https://ios.exs.lv/groups/{plus|minus}/{mb_id}

{plus|minus} - atbilstoši pozitīvam vai negatīvam vērtējumam;
{mb_id} - minibloga/komentāra/atbildes ID

Atbilde:

json
{
    ...,
    "response": {
        "vote_value": int
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
vote_value ieraksta jaunais vērtējums pēc izpildīta pieprasījuma

Lietotāja pieteikšanās grupai

Ļauj lietotājam pieteikties par biedru kādā no grupām.

Adrese: https://ios.exs.lv/groups/{group_id}/apply

{group_id} - ID grupai, kurā lietotājs vēlas kļūt par biedru

Atbilde:

json
{
    ...,
    "response": {
        "approved": int
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
approved vērtība vienmēr būs vienāda ar 1 (ja vien netiks atgriezta kāda kļūda parametrā "success" un "message", tādējādi šo parametru nemaz nepievienojot)

Lietotāja izstāšanās no grupas

Ļauj lietotājam izstāties no grupas.

Adrese: https://ios.exs.lv/groups/{group_id}/leave

{group_id} - ID grupai, no kuras lietotājs vēlas izstāties

Atbilde:

json
{
    ...,
    "response": {
        "left": int
    }
}

Atbildes vērtība skaidrota tabulā:

Parametrs Skaidrojums
left vērtība vienmēr būs vienāda ar 1 (ja vien netiks atgriezta kāda kļūda parametrā "success" un "message", tādējādi šo parametru nemaz nepievienojot)
