Skaidrojums

Šajā sadaļā aprakstīti tie pieprasījumi, kas neietilpst citās sadaļās.
Šādi pieprasījumi veicami uz adresi https://ios.exs.lv/random/.

Lai izpildītu kādu no pieprasījumiem, lietotājam jābūt autentificētam.

Lietotāja notikumi

Atgriež sarakstu ar 25 (maināma vērtība) jaunākajiem lietotāja notikumiem.

Adrese: https://ios.exs.lv/random/notifications

Atbilde:

json
{
    ...,
    "response": {
        "notifications": [
            {
                "id": int,
                "type": int,
                "group_id": int,
                "foreign_key": int,
                "text": string,
                "date": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
notifications masīvs ar objektiem par jaunākajiem notikumiem
notifications : id notikuma ID
notifications : type notikuma paveids (skat. nākamo tabulu)
notifications : group_id grupas ID, ja notikums saistīts ar kādu domubiedru grupu
notifications : foreign_key ID, kas saistīts ar konkrēto notikumu un dažādiem notikumu tipiem apzīmē ko citu (skat. tabulu)
notifications : text īss notikuma teksts, piemēram, fragments no minibloga teksta.
Neskaitot miniblogus, citiem notikumiem pārsvarā tukšs string.
notifications : date notikuma parādīšanās laiks teksta formātā, piemēram, "pirms 3 mēnešiem"

Tiešsaistē esošie lietotāji

Atgriež sarakstu ar tiešsaistē esošajiem lietotājiem pēdējās 5 minūtēs.

Adrese: https://ios.exs.lv/random/online

Atbilde:

json
{
    ...,
    "response": {
        "online": int,
        "registered": int,
        "users": [
            {
                "nick": string,
                "params": string
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
online tiešsaistē esošo lietotāju skaits
registered tiešsaistē esošo lietotāju skaits, kas ir autentificējušies
users masīvs ar objektiem par tiešsaistē esošajiem autentificētiem lietotājiem
users : nick lietotāja lietotājvārds
users : params vairākas vērtības, atdalītas ar "|", kas raksturo lietotājvārdu, lai to varētu pareizi noformatēt, piemēram, "115|2|1|3|0":
1) lietotāja ID
2) lietotāja klase: 0 - parasts lietotājs, 1 - administrators, 2 - moderators, 3 - rakstu autors, 5 - bots
3) 0 - lietotājs nav online, 1 - lietotājs ir online
4) ierīce, caur kuru lietotājs skatās lapu: 0 - exs.lv · 1 - m.exs.lv. · 2 - android app · 3 - iOS app
5) vai lietotājam ir aktīvs liegums? 0 - nav, 1 - ir

Lietotāja profila informācija

Atgriež norādītā lietotāja profila datus.

Adrese: https://ios.exs.lv/random/profile/{user_id}

{user_id} - lietotāja ID, par kuru datus atgriezt

Atbilde:

json
{
    ...,
    "response": {
        "profile": {
            "nick": string,
            "params": string,
            "avatar_url": string,
            "days_online": int,
            "days_registered": int,
            "last_seen": string,
            "usertitle": string,
            "gender": int,
            "web": string,
            "karma": int,
            "post_count": int,
            "page_count": int,
            "self_votes_count": int,
            "self_votes_sum": int,
            "other_votes_sum": int
        },
        "awards": {
            "total_count": int,
            "top_count": int,
            "list": [
                {
                    "img_url": string,
                    "title": string
                },
                ...
            ]
        }
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
profile informācija par lietotāju
profile : nick lietotāja lietotājvārds
profile : params lietotāja lietotājvārda formatēšanai nepieciešamie parametri
profile : avatar_url saite uz lietotāja avataru
profile : days_online dienu skaits, cik pēc kārtas lietotājs ir bijis tiešsaistē
profile : days_registered dienu skaits, cik ilgi lietotājs jau ir reģistrējies lapā
profile : last_seen laiks teksta formā, kad pēdējoreiz lietotājs redzēts
profile : usertitle lietotāja nosaukums profilā
profile : gender lietotāja dzimums (0 - vīrietis, 1 - sieviete)
profile : web lietotāja norādītās mājaslapas adrese
profile : karma lietotāja karma
profile : post_count lietotāja veikto ierakstu skaits lapā
profile : page_count lietotāja izveidoto rakstu skaits lapā
profile : self_votes_count cik reizes lietotājs vērtējis kādu ierakstu
profile : self_votes_sum lietotāja veikto vērtējumu summa
profile : other_votes_sum summa vērtējumiem, kādus šī lietotāja ierakstiem atzīmējuši citi lietotāji
awards informācija par lietotāja medaļām
awards : total_count lietotājam kopā piešķirto medaļu skaits
awards : top_count lietotāja svarīgāko medaļu skaits (sakrīt ar parametrā "list" esošo objektu skaitu)
awards : list masīvs ar objektiem par svarīgākajām lietotājam piešķirtajām medaļām (tām, kuras lietotājs starp visām medaļām novietojis kā pirmās sarakstā; sakrīt ar parametrā "top_count" parādīto vērtību)
awards : list : img_url saite uz medaļas attēlu
awards : list : title medaļas nosaukums

Lietotāja grupu saraksts

Atgriež sarakstu ar grupām, kurās lietotājs ir pieteicies.

Adrese: https://ios.exs.lv/random/mygroups

Atbilde:

json
{
    ...,
    "response": {
        "group_count": int,
        "groups": [
            {
                "id": int,
                "avatar_url": string,
                "title": string,
                "member_count": int,
                "post_count": int,
                "is_admin": bool,
                "is_mod": bool,
                "unread_msgs": int
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
group_count grupu skaits, kurās lietotājs ir pieteicies vai kuras ir izveidojis
groups saraksts ar grupām
groups : id grupas ID
groups : avaar_url saite uz grupas avataru
groups : title grupas nosaukums
groups : member_count grupas biedru skaits, ieskaitot grupas autoru
groups : post_count grupā veikto ierakstu skaits
groups : is_admin vai autentificētais lietotājs šajā grupā ir administrators
groups : is_mod vai autentificētais lietotājs šajā grupā ir moderators
groups : unread_msgs autentificētā lietotāja neizlasīto ierakstu skaits šajā grupā

Grupu kategoriju saraksts

Atgriež sarakstu ar grupu kategorijām.

Adrese: https://ios.exs.lv/random/gcategories

Atbilde:

json
{
    ...,
    "response": {
        "group_count": int,
        "group_categories": [
            {
                "id": int,
                "title": string,
                "group_count": int
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
group_count grupu kopskaits pa visām grupu kategorijām
group_categories masīvs ar objektiem par grupu kategorijām
group_categories : id kategorijas ID
group_categories : title kategorijas nosaukums
group_categories : group_count grupu skaits šajā kategorijā

Grupas kategorijas grupu saraksts

Atgriež sarakstu ar grupām norādītajā grupu kategorijā.

Adrese: https://ios.exs.lv/random/groups/{cat_id}?page={page_nr}

{cat_id} - grupu kategorijas ID
{page_nr} - grupu saraksta lappuse. Parametru nenorādot, atgriež pirmo lappusi.

Atbilde:

json
{
    ...,
    "response": {
        "category_id": int,
        "category_title": string,
        "groups": [
            {
                "id": int,
                "avatar_url": string,
                "title": string,
                "member_count": int,
                "post_count": int,
                "in_group": bool,
                "is_admin": bool,
                "is_mod": bool,
                "unread_msgs": int
            },
            ...
        ]
    }
}

Atbildes vērtības skaidrotas tabulā:

Atslēga Skaidrojums
category_id grupu kategorijas ID
category_title grupu kategorijas nosaukums
groups masīvs ar objektiem par kategorijas grupām
groups : in_group vai autentificētais lietotājs ir pieteicies šajā grupā?

  •    - skatāmo lappušu skaits nav ierobežots
  •    - katrā lappusē tiek atgriezta informācija par 20 grupām (maināma vērtība)
